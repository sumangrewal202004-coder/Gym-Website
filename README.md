# Gym-Website
Responsive gym website built using HTML, SCSS, and JavaScript with modern UI and smooth navigation.

# P0WER FITNESS
## About This Project

This is a responsive business website built to showcase services, provide company information, and allow users to get in touch easily. The website contains multiple pages including Home, About, Services, and Contact, with a clean and user-friendly interface.

# What I Learned

## HTML5 Structure
Learned to use semantic tags like header, nav, section, article, and footer to improve readability and accessibility.

## CSS Layouts
Implemented CSS Grid and Flexbox to create structured and responsive layouts for different screen sizes.

# SASS Organization
Organized styles efficiently using variables and modular structure to keep the CSS clean and maintainable.

## JavaScript
Added interactivity such as smooth scrolling, form alerts, and navigation behavior.

## Accessibility
Ensured proper contrast, readable fonts, and keyboard-friendly navigation for better usability.

## Challenges & Solutions

Challenge: Making the layout responsive across devices
Solution: Used CSS Grid and media queries to adapt the layout for mobile, tablet, and desktop screens.
